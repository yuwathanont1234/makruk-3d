# หมากรุกไทย 3D (Makruk 3D)

เกมหมากรุกไทยแบบ 3D ที่ตัวหมากเดินมี **Animation** จริง พร้อม:

- 🎮 **2 โหมด**: เล่น 2 คนบนเครื่องเดียว และเล่นกับ AI (ง่าย/กลาง/ยาก)
- 🎨 **เปลี่ยนสไตล์ตัวหมากได้สด ๆ**: อยุธยา · โมเดิร์น · อวกาศ · โรมัน · AI-โฮโลแกรม · คนทำงาน
- 🔊 **เปลี่ยนชุดเสียงได้** + เสียงเฉพาะของหมากแต่ละตัว
- ♟️ กติกาหมากรุกไทยครบ (เลื่อนขั้นเบี้ย, รุก, รุกจน, อับ)

## โครงสร้าง (monorepo)

```
packages/engine   @makruk/engine — กติกา + AI (TypeScript ล้วน, ใช้ร่วมทุกแพลตฟอร์ม)
apps/web          เว็บ 3D (Vite + Three.js)   ← Phase 1 (เล่นได้แล้ว)
apps/mobile       แอป Expo                      ← Phase 3
supabase/         เล่นออนไลน์ + AI สร้างธีม      ← Phase 2
```

## เริ่มใช้งาน

```bash
npm install
npm test            # ทดสอบ engine (กติกา + AI)
npm run dev         # เปิดเว็บเกม (Vite dev server)
```

เปิดเบราว์เซอร์ตาม URL ที่ Vite แสดง แล้วเล่นได้เลย

## สถาปัตยกรรมข้ามแพลตฟอร์ม

ทุก client `import "@makruk/engine"` ตัวเดียวกัน → กติกา/AI เหมือนกันเป๊ะ
เวลาเล่นออนไลน์จะ **ส่งแค่ "ตาเดิน" (move) ผ่าน network** แล้วทุกเครื่องเดินผ่าน engine ตัวเดียวกัน → state ตรงกันเสมอ
ส่วนการแสดงผล (ภาพ 3D / เสียง) เป็น layer แยก สลับธีมได้โดยไม่กระทบกติกา
